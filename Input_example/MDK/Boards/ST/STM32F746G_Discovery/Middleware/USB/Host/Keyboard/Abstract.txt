This is an USB Host HID Keyboard example.

After the USB HID keyboard is connected it displays 
pressed characters on standard output.

Board:                  ST 'STM32F746G-Discovery'
Microcontroller:        ST 'STM32F746NGH6'
Clock Settings:         XTAL       =  25 MHz
                        CPUCLK     = 216 MHz
                        USB FS CLK =  48 MHz
                        USB HS CLK =  60 MHz (from external ULPI PHY)
USB Full-speed:         OTG_FS (connector CN13)
USB High-speed:         OTG_HS (connector CN12)
User Interface:         input:  USB HID Keyboard
                        output: Graphic LCD

Detailed description is available on:
http://www.keil.com/pack/doc/MW/USB/html/host_hid_tutorial.html

The program is available for target(s):

  - STM32F746 Flash: Downloads to and executes from internal Flash

JUMPER SETTINGS:
  - JP1 in position "5V link"
